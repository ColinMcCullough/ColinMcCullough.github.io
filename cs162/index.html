CS 162 Course Description

This class will review briefly CS161 and continue programming in Java. The emphasis of this term is on the development of classes, data structures, algorithm analysis, recursion, and sorting. However we will also explore/review several basic programming constructs, Inheritance, Polymorphism, Interfaces, Exceptions, and Files/Streams.

CS 162 Learning Outcomes

  • Create proper program documentation
  • Use a subset of the Java collections classes
  • Understand and select appropriate Java collections classes
  • Design OOP software with use of interfaces, abstract classes, inheritance, and polymorphism
  • Explore basic data structures (stacks, queues, linked lists, etc.)
  • Write a simple sorting algorithm
  • Identify the Big O of simple algorithms
